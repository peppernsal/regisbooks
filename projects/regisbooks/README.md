# Regis Books

The Regis Books project aims to increase the circulation and re-use of student textbooks within Regis. The project enables students to easily lend and give physical copies of books to each other, thereby reducing the amount of money that is spent on buying textbooks each year and reducing the amount of textbooks that inevitably go out of use early.

This project consists of a website to share books and a client library (Python, C#) to use the data programatically within code.
